## required library to install 
1. express
    - npm install express
2. cors
    npm install cors
3. superagent
    - npm install superagent
4. method override
    - npm install method-override
5. ejs 
    - npm install ejs
6. nodemon
    - npm install -g nodemon
7. session 
    - npm install express-session
8. bcrypt
    - npm install bcrypt
